Smarttrak Alarma

Плагин для WordPress.

Что делает

- Добавляет кнопку обратного звонка на сайт.
- - Открывает popup с формой.
  - - Принимает имя и телефон.
    - - Делает телефон обязательным.
      - - Отправляет заявку на почту владельца.
        - - Отправляет аларму в Telegram.
          - - Ловит отправку формы контактов SureForms и тоже шлет аларму в Telegram.
           
            - Заголовки Telegram
           
            - - Для обратного звонка:
              -   НОВАЯ ПОЗВОНИ МНЕ АЛЯРМА!!!
              -   - Для формы контактов:
                  -   НОВАЯ ФОРМА В КОНТАКТАХ АЛЯРМА!!!
               
                  -   Что входит
               
                  -   - Плавающая кнопка.
                      - - Popup-форма.
                        - - Настройки в админке WordPress.
                          - - Кнопка для подтягивания chat ID из Telegram-бота.
                            - - Шорткод:
                              -   [smarttrak_alarma_button]
                             
                              -   Установка
                             
                              -   1. Загрузить плагин в WordPress.
                                  2. 2. Активировать его.
                                     3. 3. Открыть настройки:
                                        4.    Settings > Smarttrak Alarma
                                        5.4. Указать или проверить:
                                             - Bot Token
                                             - Chat ID
                                             - ID формы контактов
                                             - Почту владельца
                                          5. Если Chat ID пустой:
                                          6.    - открыть бота в Telegram
                                                -    - нажать Start
                                                     -    - вернуться в настройки плагина
                                                          -    - нажать "Підтягнути Chat ID з бота"
                                                           
                                                               - Как работает обратный звонок
                                                           
                                                               - 1. Клиент нажимает кнопку.
                                                                 2. 2. Открывается popup.
                                                                    3. 3. Клиент оставляет имя и телефон.
                                                                       4. 4. Плагин:
                                                                          5.    - шлет письмо владельцу
                                                                                -    - шлет Telegram-аларму
                                                                                 
                                                                                     - Как работает форма контактов
                                                                                 
                                                                                     - 1. Клиент отправляет форму SureForms.
                                                                                       2. 2. Плагин проверяет ID формы.
                                                                                          3. 3. Если это нужная форма, отправляет Telegram-аларму с полями формы.
                                                                                            
                                                                                             4. Настройки по умолчанию
                                                                                            
                                                                                             5. - ID формы контактов: 1793
